# Roulette Monte Carlo Simulation and Convergence Test

## Overview

This is a Python based simulation of a European roulette game using Monte Carlo methods, running 1,000,000 games which all bet on the same number the user inputs. The simulation gets a convergence graph of the estimated probability of winning over the games, the probability of a win itself after all 1,000,000 games, and an estimate of the house edge.

## Key Concepts

- RNG
- Statistical analysis of probabilities as the sample size increases (Law of Large Numbers)
- Graphing convergence

## How to Run

1. Download or clone this repository
2. Run in the terminal: pip install -r requirements.txt
3. Run the simulation: python roulette_simulation.py
